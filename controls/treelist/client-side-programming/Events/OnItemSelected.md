# OnItemSelected


This event is fired when a TreeListDataItem is selected on the client.



| | |
| ------ | ------ |
| **Fired by** | RadTreeList |
| **Arguments** | get_item() - the selected TreeListDataItem object |
| **Can be canceled** | No |
