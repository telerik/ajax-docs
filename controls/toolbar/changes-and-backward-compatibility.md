# Changes and backward compatibility





## RadToolBar for ASP.NET AJAX Q2 2010

Since Q2 2010 all major changes for the controls are listed in the official Release Notes posted [here](http://www.telerik.com/products/aspnet-ajax/whats-new/release-history.aspx).

## RadToolBar for ASP.NET AJAX Q1 2010 (Version Number: 2010.1.309)

* RadToolBar for ASP.NET AJAX is fully backward compatible with its previous version

## RadToolBar for ASP.NET AJAX Q3 SP2 2009 (Version Number: 2009.3.1314)

* RadToolBar for ASP.NET AJAX is fully backward compatible with its previous version

## RadToolBar for ASP.NET AJAX Q3 SP1 2009 (Version Number: 2009.3.1208)

* RadToolBar for ASP.NET AJAX is fully backward compatible with its previous version

## RadToolBar for ASP.NET AJAX Q3 2009 (Version Number: 2009.3.1103)

* RadToolBar for ASP.NET AJAX is fully backward compatible with its previous version

## RadToolBar for ASP.NET AJAX Q2 2009

* RadToolBar for ASP.NET AJAX is fully backward compatible with its previous version

## RadToolBar for ASP.NET AJAX Q1 2009

* Three skins have been removed - Gray, Inox and SkyBlue.

* Design-time code is placed in a new assembly - Telerik.Web.Design.dll. Read [this blog post](http://blogs.telerik.com/atanaskorchev/posts/09-03-06/Meet_Telerik_Web_Design_dll.aspx) for more information.

## RadToolBar for ASP.NET AJAX Q3 2008 (Version Number 2008.3.1105)

RadToolBar for ASP.NET AJAX Q3 2008 is fully backward compatible with its previous version (Q2 2008)



## RadToolBar for ASP.NET AJAX Q2 2008 (Version Number 2008.2.723)

RadToolBar for ASP.NET AJAX Q3 2008 is fully backward compatible with its previous version (Q1 2008)
