# WCAG 2.0 and Section 508 Accessibility Compliance



## 

The interface of __RadColorPicker for ASP.NET AJAX__ is level AAA accessible (in compliance with the W3C Web Accessibility Guidelines 2.0) as well as Section 508 compliant. It passes the check of the [WAVE](http://wave.webaim.org/) automated content compliance tool for Section 508 and WCAG 2.0 - Compliance Level AAA.

>note If the __CaptchaImage-EnableCaptchaAudio__ property is set to __"true"__ the Captcha compliance level is A.
>


It also offers [Localization]({%slug captcha/accessibility-and-internationalization/localization%}).

The control is also visible in Windows High Contrast mode.
