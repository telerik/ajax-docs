# Telerik.Web.UI.RadTabBinding

Defines the relationship between a data item and the tab it is binding to in a 
            	Telerik.Web.UI.RadTabStrip control.

## Inheritance Hierarchy

* Telerik.Web.UI.RadTabBinding
